# Security policy

## Reporting a vulnerability

Do not open a public issue for suspected vulnerabilities or leaked credentials. Use GitHub's private security advisory form:

https://github.com/jiezeng2004-design/PatchWarden/security/advisories/new

Include the affected version, a minimal reproduction, impact, and suggested mitigation. Remove real keys, tokens, cookies, local paths, and private task data before submitting.

## Supported versions

Security fixes target the latest published `patchwarden` version. The pre-rename package is frozen and receives no new feature releases.
